# Programa-em-python-que-baixa-videos
Programa em Python para baixar vídeos ou playlists com histórico de downloads e interface de seleção de pasta.
## Funcionalidades
- Download de vídeos
- Download de playlists
- Escolha da pasta para salvar
- Histórico de downloads
- Barra de progresso no terminal
## Tecnologias usadas
- Python
- yt-dlp
- Tkinter
## Como usar
1. Instale as dependências:
pip install -r requirements.txt
2. Execute:
python downloader.py
3. Cole o link do vídeo ou playlist.
